# oehu-website

The website of OEHU, oehu.org.

## Project setup

Install `npm` & `node`

Use node v9: `nvm use 9.0.0`

Install dependencies:

    npm install

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
